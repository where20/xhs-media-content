# 冷启动期 SOP · 0 粉 0 互动 时的复盘 + 调整 + 运营机制

> **适用阶段：** 第 1-30 天（新号 / 0-100 粉 / 0 评论或个位数评论）
> **核心原则：** 过程指标代替结果指标——数据没来之前，看你能控制的东西。
> **读完时间：** 15 分钟 | **执行频率：** 每天 30-60 分钟

---

## 1. 重新定义"复盘"——别看数据，看过程

> ❌ **不要做的事：** 发完一篇就刷后台看曝光/点赞/评论，5 分钟看 8 次。
> ✅ **要做的：** 把"复盘"换成 **发布前自检** + **24h 过程指标**，跟数据脱钩。

### 1.1 发布前自检清单（每篇必过 3 项）

| # | 检查项 | 怎么测 | 通过标准 |
|---|--------|--------|----------|
| 1 | **封面 3 秒测试** | 让 2-3 个朋友看封面 3 秒，问"这篇讲啥" | 3 个里有 2 个说对 |
| 2 | **标题公式** | 套 `人群 + 场景 + 数字/悬念` 三选二 | 缺一项就重写 |
| 3 | **互动钩子** | 文末必须有 `评论区扣 1` / `A or B` / `你的故事` 之一 | 没钩子直接重发 |

> 📌 这 3 项**比数据重要**——封面 + 标题决定"看不看"，钩子决定"留不留"。

### 1.2 24h 复盘表（推荐填到 daily-content.json 的 `selfCheck` 字段）

```json
{
  "day": "D4",
  "selfCheck": {
    "cover_3s_test": "pass",            // pass / fail
    "title_formula": "pass",            // pass / fail
    "interaction_hook": "pass",         // pass / fail
    "6h_exposure_check": "进入二级池",    // 详见 §3
    "improvements": "标题第 2 版太抽象，改成'32岁前端 · 30天跑了 20 个 AI 工具'具体了",
    "next_day_pivot": "封面 v2 改大字 + 数字"
  }
}
```

## 2. 4 个动态调整策略（按阶段触发）

| 阶段 | 信号 | 调整动作 | 千万别做 |
|------|------|---------|---------|
| **D1-D3** 验证内容方向 | 写得卡不卡？读完自己的笔记想转发吗？ | **卡 → 当晚就改选题方向**，别硬扛 30 天 | 死磕一个方向，30 天全废 |
| **D4-D10** 验证封面/标题 | 朋友 3 秒测试结果；标题 3 选 1 投票 | **3 秒说不清 → 改封面**；标题投票输 → 用赢的套路批量改 | 自己 YY 觉得好就行 |
| **D11-D20** 验证"种子互动" | 朋友/同事/私域群有没有人**主动转**或**评**？ | **0 种子互动 → 内容离他们太远**，回 D1-D3 重选人群 | 继续日更，但不调整方向 |
| **D21-D30** 验证"能不能熬" | 这阶段你应该有 50-200 真粉 | **没到 → 别继续日更**，降到 3 天 1 篇，精力分给研究头部 + 复盘 | 焦虑到日更 2 篇，质量崩 |

### 2.1 调整的本质：换方向，不是更努力

> 调整 = **重新选择**（选题 / 人群 / 内容支柱 / 封面风格）
> 不是 = 加大发布频率 / 抄更"爆"的标题 / 买流量

**触发"换方向"的 3 个红灯**：
1. 写到第 3 篇你开始**不想写**了
2. 朋友看了你 3 篇笔记**问不到一句"所以你想表达啥"**
3. 你的目标人群（朋友/同事/私域群）**没一个人主动转发**

3 个红灯中 2 个亮 → 当晚复盘 + 第 2 天换方向。

## 3. 6h 曝光自查——唯一该看的"数据"

> **算法判定逻辑：** 发布后 30 分钟给你"一级流量池"（约 200-500 曝光）。如果 6 小时内**互动率**（点赞+收藏+评论）/ 曝光 ≥ 3%，进入"二级流量池"（约 2000-5000 曝光）。否则笔记沉底。

### 3.1 自查步骤

| 时间 | 动作 | 怎么记 |
|------|------|--------|
| 发布后 30 分钟 | 看一次曝光数（记为 A） | 写下来或截图 |
| 发布后 6 小时 | 再看一次曝光数（记为 B） | 算 `B / A` |

**判定**：
- `B / A ≥ 2` → ✅ **进入二级池**，按原计划继续
- `B / A < 2` → ❌ **没进二级池**，见 §3.2

### 3.2 没进二级池怎么办（24h 内立刻救）

按优先级做，**别等明天**：

1. **改标题**（最有效）— 编辑原笔记标题，套 §1.1 的公式
2. **改封面**（次有效）— 换 1 张视觉对比更强的图
3. **自己发评论** — 写一条**有立场的**评论（不是"写得真好"）
4. **私域发 2-3 个群** — 请真人评论（不是点赞）

> ⚠️ **小红书规则：** 一篇笔记 24h 内可编辑 2-3 次，不会限流。超过 5 次编辑会降权。

### 3.3 6h 曝光检查表（建议加到 daily.html 的 24h SOP 卡片里）

```
笔记：___________
发布 30 分钟曝光（A）：___________
发布 6 小时曝光（B）：___________
B / A = ___________

□ ≥ 2 → 进入二级池，继续
□ < 2 → 触发 §3.2 救援
```

## 4. 5 个"无数据"也能用的运营动作

> 这些都是**自己干**就能产生互动，0 粉也能起手。

### 4.1 自己评论扣 1（每篇必做）

- 发布后 5 分钟内**立刻**自己评论，占前 3 条评论位
- 评论内容**必须有立场**（不是"首评"），例如：
  - ❌ "首评 / 自留"
  - ✅ "我先来：跑了 3 套 AI 流程，最意外的是 B 组——0 事实错误但开头钩子弱"
- **作用：** 营造"这篇有讨论"的氛围，陌生人来时不容易秒关

### 4.2 互赞联盟 5-10 人（D1 就建）

- 找**同赛道** 5-10 个 0-500 粉的号（小红书/即刻都算）
- 每天互相访问 + **认真评论**（**别刷赞**——刷赞会被限流）
- 群的作用是**互相给真实评论**，评论里有"立场"才能让算法识别成"互动"

**互赞联盟名单模板**：

| 账号 | 平台 | 粉丝 | 赛道 | 上次互访 | 备注 |
|------|------|------|------|----------|------|
| @xxx | 小红书 | 230 | AI 电商 | 2026-06-24 | 主写 AI 文案 |
| @yyy | 小红书 | 89 | 程序员副业 | 2026-06-24 | 同转型阶段 |

### 4.3 抢同赛道头部评论区前排（每天 30 分钟）

- 头部爆了立刻去评论
- 评论**带自己的观点 + 自己内容的钩子**（不是"写得真好"）
- 例：
  - ❌ "学到了！"
  - ✅ "你这个流程我跑过，A 组的'一句话直接'实际改稿成本反而最高——我在 D2 测过同一款保温杯 3 套 AI 流程，B 组 0 错误改了 1 处"

### 4.4 私域首发（D1 就开始）

- 每篇笔记链接发到 **3-5 个微信群/朋友圈**
- 请**真人评论**（不是点赞）——评论里的关键词会被算法识别
- **真人评论 = 算法判定"这篇有讨论价值"的最强信号**

### 4.5 6h 曝光自查（每篇必做）

- 详见 §3，别跳过
- **6h 不动 = 24h 内必须改**，别等明天

## 5. 30 天节奏表（不焦虑版）

| 天 | 任务 | 复盘重点 | 关键检查 |
|----|------|---------|---------|
| D1 | 发 + 找 5 个互赞人 + 私域发 3 个群 | 你写得爽不爽 | 钩子有没有 |
| D2 | 复盘 D1 封面 + 标题 | 朋友 3 秒测试 | title_formula |
| D3 | 按 D2 调整发 D3 | 6h 曝光自查 | 6h_exposure_check |
| D4-D7 | 每天 1 篇 + 每天 30 分钟评论同赛道头部 | 种子互动有没有 | 私域评论数 |
| D8 | 周末复盘：4 篇里哪 1 篇最想写？最不痛苦？ | **找到方向感** | 想不想继续 |
| D9-D10 | 按 D8 调整方向 + 发 2 篇 | 进入节奏 | 写作耗时下降 |
| D11-D20 | 日更 + 抢评论区 + 私域分发 | 内容质量稳定 | 收藏率 ≥ 3% |
| D21 | 中期复盘 | 50 粉没到？ | 见 §2 表格 |
| D22-D30 | 按 D21 决策：日更 or 降频 | 别硬扛 | 涨粉速度 |

### 5.1 30 天没粉的退路

> **别硬扛 30 天，21 天没感觉就降频。**
> 日更 1 篇 30 天 = 30 篇废稿
> 不如 3 天 1 篇 10 篇精品 + 复盘迭代

**降频信号**（D21 触发任一即降频）：
- 50 粉目标差 ≥ 50%
- 你开始**不想写**
- 私域群 0 真人评论超过 5 篇
